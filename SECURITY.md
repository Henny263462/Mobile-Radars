# Security

## Supported versions

Security fixes are applied to the **latest release** on the default branch for Minecraft **1.21.1** / NeoForge, unless noted otherwise in release notes.

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security-sensitive problems (exploits, RCE, credential leaks in configs, etc.).

Instead:

1. Open a **private** report via [GitHub Security Advisories](https://github.com/Arsenalists-of-Create/Create-Radar/security/advisories/new) if this project lives in that repository, **or**
2. Contact the maintainers through the [Create Arsenal Discord](https://discord.gg/Cyp7GnSCZG) with a clear description and reproduction steps.

Include Minecraft / NeoForge / mod versions and any relevant logs. We will acknowledge receipt and work on a fix or mitigation as soon as practical.

## Scope

This policy covers **Create Radar: Mobile Radars** only. Vulnerabilities in **Create**, **Create: Radars**, **NeoForge**, or **Minecraft** should be reported to those projects.
